Pra que serve a Tag Head ?

O header elemento representa um contêiner para conteúdo introdutório ou um conjunto de links de navegação. Um header elemento normalmente contém: um ou mais elementos de título (h1 - h6) logotipo ou ícone informações de autoria.

Nota: Você pode ter vários header elementos em um documento HTML. No entanto, header não pode ser colocado dentro de um footer , address ou outro header elemento.
